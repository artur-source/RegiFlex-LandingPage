# RegiFlex - Landing Page

Landing page corporativa e comercial para a startup **RegiFlex**, um sistema de organização de fluxo de atendimento clínico.

## 🎯 Sobre o Projeto

RegiFlex é uma plataforma SaaS que digitaliza e organiza o fluxo de atendimento em clínicas e consultórios, desde a chegada do paciente até o fim do atendimento. A landing page apresenta a empresa, o produto, suas funcionalidades, diferenciais competitivos e informações de contato.

## 📋 Conteúdo da Landing Page

### Seções Implementadas

1. **Header** - Navegação responsiva com menu mobile
2. **Hero** - Apresentação principal com CTAs persuasivas
3. **About** - Quem somos, história, propósito e identidade visual
4. **Product** - Descrição da solução, público-alvo e plataformas
5. **Features** - Principais funcionalidades e fluxo de uso
6. **Differentials** - Diferenciais competitivos e comparativo com concorrentes
7. **HowToAccess** - Processo de onboarding, planos e preços
8. **Team** - Organograma da equipe e estrutura organizacional
9. **Contact** - Formulário de contato e informações
10. **Footer** - Links rápidos, redes sociais e políticas

## 📦 Instalação e Execução

### Pré-requisitos
- Node.js 18+
- npm ou pnpm

### Passos

1. **Clone o repositório:**
   ```bash
   git clone https://github.com/artur-source/RegiFlex-LandingPage.git
   cd RegiFlex-LandingPage
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

4. **Acesse no navegador:**
   ```
   http://localhost:5173
   ```

## 📝 Scripts Disponíveis

```bash
# Desenvolvimento
```
